
# Changelog

All notable changes to this project will be documented in this file.

The format is based on "Keep a Changelog" and this project adheres to semantic versioning.

## [1.0.0] - 2026-06-07
### Added

- Initial release of WedPlan Mobile App.
- Role-based dashboards for `couple` and `vendor` users.
- Couple features: budget tracking, expenses, tasks, guest list, vendor browsing.
- Vendor features: service management, booking overview, booking actions.
- Guest flow: invitation lookup, QR/code input, RSVP and check-in flow.
- Push notifications support via Firebase Messaging.
- Session persistence using secure local storage.

### Notes

- Backend API required for full functionality. Default API: `https://wedplan.projectse.io/api/v1`.
- Some platform-specific setup (Firebase configuration) is required for notifications.
